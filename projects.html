Go Microservices CI/CD with GitOps on AWS EKS

Engineered an advanced **GitOps** pipeline for a Go microservice using **Argo CD** for declarative, automated deployments on an **AWS EKS** cluster. This project leverages **Infrastructure as Code (IaC)** with **Terraform** and a **two-repository model** for a secure and self-healing workflow.

  • **Self-Healing Deployment:** Argo CD continuously monitors the cluster, automatically correcting any drift from the desired state in Git.
  • **Streamlined Workflow:** Automated CI pipeline with **GitHub Actions** runs tests, security scans, and triggers the deployment without any manual intervention.
  • **Full Automation:** From a single `git push` to a live update on Kubernetes, the entire process is automated and auditable.

Technologies Used:

  • **IaC/Orchestration:** Terraform, Kubernetes, Helm
  • **CI/CD:** GitHub Actions, Argo CD, GitOps
  • **Cloud:** AWS EKS, AWS VPC
  • **Security:** Gitleaks, Trivy
  • **Languages:** Go, Docker
